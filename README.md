# GHBackup
The program is designed to quickly clone public repositories from GitHub. After launching, specify the `Login`, `Working directory` and click `Backup`. As a result, a shallow clone of all repositories and the final archive will be obtained: `~/WorkDir/1-BACKUP/GitHub.tar.gz`.  
  
**Dependencies:** git curl tar gzip gtk2
